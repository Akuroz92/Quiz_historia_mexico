# Quiz de Historia de México

Juego interactivo de preguntas y respuestas sobre la Independencia y Revolución Mexicana.

## Cómo ejecutarlo

1. Asegúrate de tener **Python** instalado en tu computadora.
2. Descarga los archivos de este repositorio.
3. Coloca el archivo `preguntas.csv` en la misma carpeta que `quiz_historia_mexico.py`.
4. Ejecuta el programa haciendo doble clic sobre `quiz_historia_mexico.py`.
